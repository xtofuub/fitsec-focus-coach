# Fitsec Focus Coach
### Ultra-Refined Precision Productivity

Fitsec Focus Coach is an elite focus management system designed for high-performance professionals. Built with the **Glass Obsidian** design language, it combines advanced productivity logic with a premium, immersive aesthetic to eliminate distractions and maximize cognitive output.

---

## 💎 Design Philosophy: Glass Obsidian

The application utilizes a dark-first, glassmorphism-driven interface that feels alive and responsive.
- **Surface Geometry**: 16px radius corners with subtle 1px obsidian borders.
- **Atmospheric Glow**: Neon Cyan (#00FFFF) accents with amber and emerald feedback states.
- **Dynamic Layers**: 24px-32px backdrop blurs with 0.02 opacity noise overlays for a tactile, cinematic feel.
- **Typography**: Precision Geist Variable and Monospace tabular font pairings for maximum legibility.

---

## 🚀 Key Features

### 🧩 Interactive Cycle Blueprint
Break away from static timers. The Focus Coach features a visual timeline editor where you can craft your entire mission before launching.
- **Segment Toggling**: Instantly swap between Short and Long breaks by clicking the blueprint.
- **Dynamic Sequence**: Expand or contract your cycle with real-time total duration feedback.
- **Human-Readable Readouts**: Live total mission time (e.g., "2h 15m") calculated instantly.

### ⏱️ Precision Engine
- **End-of-Cycle Detection**: Intelligent phase-end logic that triggers a reflection session once the manual mission is complete.
- **Auto-Start Logic**: Granular control over Work and Break transitions.
- **Dual-Input Timing**: Smooth sliding stepper animations with direct numeric keyboard entry.

### 📊 Performance Intelligence
- **Deepwork Reflections**: "Nailed it?" prompts that log qualitative mission success.
- **Granular History**: Individual session management with per-session deletion and daily summary metrics.
- **Compact Sidebar Navigation**: Tooltip-supported navigation for effortless view transitions.

## 🛠️ Technical Architecture

Fitsec Focus Coach is a robust **Electron** desktop application built on a modern high-performance stack:
- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Animation**: [Framer Motion](https://www.framer.com/motion/)
- **Shell**: [Electron](https://www.electronjs.org/)
- **Styling**: Vanilla CSS (Fitsec Custom Core) + [Tailwind](https://tailwindcss.com/)
- **Iconography**: [Lucide React](https://lucide.dev/)

## 📦 Installation & Deployment

### Prerequisites
- Node.js (v18+)
- npm

### Development
Launch the development environment with Electron and Vite HMR:
```bash
npm run electron:dev
```

### Build
Generate a production-ready installer:
```bash
npm run electron:build
```
